## 📖 About

SmartYatra is a comprehensive bus booking application that simplifies the process of searching, booking, and managing bus tickets. Built with Flutter, it offers a seamless cross-platform experience with a modern, intuitive UI.

---

## ✨ Key Features

### 🔐 Authentication
- Phone & Email login with OTP verification
- Secure password management

### 🎫 Booking System
- Search buses by location and date
- Interactive seat selection
- Multiple payment methods
- E-ticket generation with QR code

### 💳 Wallet & Payments
- Digital wallet with balance management
- Transaction history
- Secure payment gateway integration

### 🗺️ Navigation & Tracking
- Real-time bus tracking on map
- Route visualization
- ETA calculation

### 💬 Support
- Live chat support
- FAQ section
- SOS emergency contact

### 🌍 Additional Features
- Multi-language support (6 languages)
- Ticket history & cancellation- Reviews & ratings

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Laxmi-02/smart_yatra.git

# Navigate to project
cd smart_yatra

# Install dependencies
flutter pub get

# Run the app
flutter run -d chrome
```
