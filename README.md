# 🚌 SmartYatra - Bus Booking App

A complete bus booking application built with Flutter.

## ✨ Features

- ✅ Login/OTP (Phone & Email)
- ✅ Bus Search & Booking
- ✅ Seat Selection
- ✅ Payment Flow
- ✅ Ticket Management
- ✅ Digital Wallet
- ✅ Live Chat Support
- ✅ Bus Tracking (Map)
- ✅ Multi-language Support
- ✅ SOS Emergency

## 🛠️ Technologies

- Flutter
- Dart
- Material Design 3
- OpenStreetMap

## 📦 Installation

```bash
git clone https://github.com/Laxmi-02/smart_yatra.git
cd smart_yatra
flutter pub get
flutter run -d chrome
```
