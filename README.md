# 🚌 SmartYatra - Bus Booking App

A complete bus booking application built with Flutter. This app allows users to search buses, select seats, make payments, and track their bus in real-time.

## ✨ Features

- ✅ **Login/OTP** - Phone & Email authentication
- ✅ **Bus Search** - Search by location and date
- ✅ **Seat Selection** - Interactive seat layout
- ✅ **Payment Gateway** - Secure payment flow
- ✅ **Ticket Management** - E-tickets with details
- ✅ **Digital Wallet** - Add money & transaction history
- ✅ **Live Chat** - Customer support system
- ✅ **Bus Tracking** - Real-time map tracking
- ✅ **Multi-language** - Support for 6 languages
- ✅ **SOS Emergency** - Quick emergency contact

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **UI:** Material Design 3
- **Maps:** OpenStreetMap (flutter_map)
- **State Management:** setState
- **IDE:** VS Code

## 📦 Installation

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Laxmi-02/smart_yatra.git

# 2. Navigate to the folder
cd smart_yatra

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run -d chrome
```
